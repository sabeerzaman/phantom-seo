What was used to make this site "crawlable"?

BackboneJS:
For routing between the pages and rendering the content on the front end.
PhantomJS:
A "headless web browser" that can be run programmatically to fetch HTML pages, and run any JavaScript required to render it fully.
NodeJS:
Platform able to run JavaScript on the server side, which is used here to handle any "escaped fragment" requests made by the crawler to run use PhantomJS to fully render and return the HTML pages.
Apache Web server:
Mainly to serve up the single HTML page and JavaScript required. Additionally, used to proxy "escaped fragment" requests to the Node server running on the same machine.
